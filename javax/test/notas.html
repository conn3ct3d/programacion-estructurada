Java Swing Documentation

1. Creating a Basic Swing Application

import javax.swing.*;

public class BasicSwingApp {
    public static void main(String[] args) {
        SwingUtilities.invokeLater(() -> {
            JFrame frame = new JFrame("Basic Swing App");
            frame.setSize(400, 300);
            frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
            frame.setVisible(true);
        });
    }
}

2. Common Swing Components

(a) JFrame – Main Window

JFrame frame = new JFrame("My Window");
frame.setSize(500, 400);
frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
frame.setVisible(true);

(b) JButton – Button

JButton button = new JButton("Click Me!");
frame.add(button);

button.addActionListener(e -> System.out.println("Button Clicked!"));

(c) JLabel – Label

JLabel label = new JLabel("Hello, Swing!");
frame.add(label);

(d) JTextField – Input Field

JTextField textField = new JTextField(20);
frame.add(textField);

(e) JPanel – Container for Components

JPanel panel = new JPanel();
panel.add(new JButton("OK"));
panel.add(new JButton("Cancel"));
frame.add(panel);

(f) JOptionPane – Simple Dialog Boxes

JOptionPane.showMessageDialog(frame, "Hello, this is a message box!");

3. Layout Managers

JPanel panel = new JPanel();
panel.setLayout(new FlowLayout());

4. Event Handling

button.addActionListener(e -> System.out.println("Button Clicked!"));

5. Running the GUI on the Event Dispatch Thread (EDT)

SwingUtilities.invokeLater(() -> {
    JFrame frame = new JFrame("My GUI");
    frame.setVisible(true);
});

Conclusion

The javax.swing package provides an easy way to build GUI applications in Java. However, for modern applications, JavaFX (javafx.*) is often recommended as it has better styling and performance.
